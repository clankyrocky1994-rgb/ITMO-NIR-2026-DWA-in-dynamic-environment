# Mobile Manipulator Pick-and-Place MuJoCo TidyBot
## Цель и постановка задачи

разработать алгоритм управления мобильным манипулятором TidyBot с голономным приводом, для выполнения операции по перемещению заготовок (небольшой вал) из одного лотка в другой.
Задача заключается в реализации полного цикла управления роботом в симуляции, включая навигацию базы, захват объекта манипулятором и координацию действий, и обход препятствий.
#Методы и алгоритмы решения
Управление поведением робота реализуется с помощью метода FSM, в котором каждое состояние соответствует этапу задачи (навигация к объекту, захват, транспортировка,размещение). 
Для движения мобильной базы используется алгоритм следования по точкам маршрута, полученного методом A*, с управлением на основе ошибки по расстоянию
## Интсрукция по запуску
### Для установки mink
```
uv add mink
```
```
pip install -r requirements.txt
```
Для запуска программы 
```
uv run examples/main.py
```
